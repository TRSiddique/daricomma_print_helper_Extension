# দাঁড়িকমা প্রিন্ট হেল্পার - Chrome Extension

## ইনস্টলেশন পদ্ধতি

### ধাপ ১: ফাইল Extract করুন
ZIP ফাইলটি যেকোনো ফোল্ডারে Extract করুন।
(ফোল্ডারটি মনে রাখুন, পরে দরকার হবে)

### ধাপ ২: Chrome এ লোড করুন
1. Chrome browser খুলুন
2. Address bar এ টাইপ করুন: `chrome://extensions`
3. উপরে ডানদিকে **"Developer mode"** টগল ON করুন
4. **"Load unpacked"** বাটনে ক্লিক করুন
5. আপনার Extract করা ফোল্ডারটি সিলেক্ট করুন
6. Extension টি যোগ হয়ে যাবে ✅

### ধাপ ৩: ব্যবহার করুন
1. `daricomma.com` এ যান
2. প্রশ্নের পেজে যান (যেখানে প্রশ্নগুলো দেখা যাচ্ছে)
3. Chrome toolbar এ Extension আইকনে ক্লিক করুন
4. পরীক্ষার শিরোনাম লিখুন
5. **"প্রিন্ট উইন্ডো খুলুন"** বাটনে ক্লিক করুন
6. নতুন উইন্ডোতে **"প্রিন্ট করুন"** বা Ctrl+P চাপুন

## টিপস
- PDF বানাতে: Print এ "Save as PDF" সিলেক্ট করুন
- দুই কলাম: ছোট প্রশ্নের জন্য "দুই কলামে সাজাও" চেক করুন
- পপআপ ব্লক হলে: daricomma.com এর জন্য পপআপ allow করুন
